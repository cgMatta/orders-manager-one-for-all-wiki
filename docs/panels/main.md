# Main Panel

The Main Panel is docked to the left side of the chart and is always visible.
It gives you a complete overview of your account and quick access to all actions.

---

## Account Card

Shows real-time account information:

- **Balance** — total account balance
- **Equity** — balance including open P&L
- **Spread** — current spread in points
- **Mode** — account margin mode (Hedging or Netting)
- **Leverage** — account leverage ratio

---

## Total Summary

Shows the current state of your positions on the active chart symbol:

- **Open positions** — number of open trades
- **Pending** — number of pending orders
- **P&L Total** — combined unrealized profit/loss

---

## Quick Actions

| Button | Action |
|--------|--------|
| **Close All** | Closes all open positions and optionally cancels pending orders |
| **BE All** | Moves the SL of all open positions to breakeven |
| **25% / 50% / 75%** | Partially closes all open positions at the selected ratio |
| **EO** | Opens / closes the Easy Order panel |
| **LO** | Opens / closes the Order List panel |

---

## OPTIONS Section

The OPTIONS section is collapsible. Click the **▲/▼** button next to the
OPTIONS header to expand or collapse it.

### General

| Option | Description |
|--------|-------------|
| **Language** | Switch the panel language between EN / ES / PT / RU |
| **Include Pending Orders on Close All** | When ON, Close All also cancels pending orders |
| **Include Commissions with Breakeven** | When ON, BE All adds a commission offset to the breakeven price |

### EasyOrder (EO)

| Option | Description |
|--------|-------------|
| **EO Labels: left side** | When ON, line labels appear on the left side of the chart |

### Order List (LO)

| Option | Description |
|--------|-------------|
| **Include Commissions in Risk** | When ON, commission is included in the risk calculation |
| **Reverse Mode** | Controls SL placement when using Breakeven on a reversed position |

---

## Dragging

The panel can be moved anywhere on the chart by clicking and dragging
the **header area**. This works in both expanded and minimized states.

---

## Minimize

Click the **−** button in the top-right corner to minimize the panel.
When minimized, the panel collapses into a small **mini menu** that shows:

| Button | Action |
|--------|--------|
| **+** | Expand the panel back to full size |
| **EO** | Open / close the Easy Order panel |
| **LO** | Open / close the Order List panel |

The mini menu can also be **dragged** to any position on the chart,
so you can keep it out of the way while still having quick access to all panels.

!!! tip
    Even with the Main Panel minimized you have full access to Easy Order
    and Order List via the mini menu — no need to expand first.
