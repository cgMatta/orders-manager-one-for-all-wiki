# Orders Manager One For All

**Orders Manager One For All** is a professional all-in-one trading panel for MetaTrader 5, designed for both retail traders and prop firm environments, it also supports cent accounts for either Netting or Hedging.

It replaces the default MT5 order interface with a modern, CCanvas-based UI that gives you full control over risk management, order placement, and position monitoring — all from a single panel.

---

## Key Features

- **Multi-panel UI** — Main Panel, Easy Order (EO) and Order List (LO) work together seamlessly
- **Risk-based sizing** — calculate lot size automatically from % risk, fixed amount or manual lots
- **Multi-TP support** — up to 4 take profits per order with individual % allocation (Hedging accounts)
- **Synthetic Netting Ladder** — multi-TP simulation for Netting accounts via partial closes
- **One-Click mode** — instant market orders with predefined SL/TP and volume
- **Trade Badges** — visual SL/TP/Entry overlays on the chart, fully interactive
- **Breakeven automation** — move SL to entry with optional commission offset
- **Bulk actions** — close all, partial close (25/50/75%), breakeven all in one click
- **Multi-language** — interface available in English, Spanish, Portuguese and Russian
- **Broker-agnostic** — works on Standard, Cent, Hedging and Netting accounts

---

## Panels Overview

| Panel | Purpose |
|-------|---------|
| **Main Panel** | Account info, quick actions, options |
| **Easy Order (EO)** | Visual order placement with draggable SL/TP lines |
| **Order List (LO)** | Live position and pending order management |

---

## Requirements

- MetaTrader 5 (build 3000+)
- Any broker account type (Standard, Cent, Hedging, Netting)
- Tested on Hedging and Netting accounts
